Ressources

Icones et illustration

Les icones et illustrations utilisées sur notre site internet proviennent de la plateforme : Icons8. Nous remercions les artistes pour leur travail exceptionnel.

Effets sonores

Les effets sonores utilisés sur notre site internet proviennent de la plateforme : pixabay. Nous remercions les artistes pour leur travail exceptionnel.

Dictionnaire

La recherche de mots est alimenté par l'API de Collins Dictionnary.

Base de données

Notre base de données est hébergée par Hostinger en France.

Elle est propulsée par MySQL.

Hostinger International Ltd. est un fournisseur de services d'hébergement web et de domaine. Il a été fondé en 2004 et est basé à Kaunas, en Lituanie. L'entreprise fournit des services d'hébergement web, de messagerie et de domaine.

Framework

Notre site internet est propulsé par Flask, un framework web open-source écrit en Python.

Service d'hébergement

Notre site internet est hébergé par Google Cloud à Francfort.

Google Cloud est une suite de services cloud proposée par Google. Elle propose des services de calcul, de stockage de données, d'analyse de données et de machine learning.

Service de messagerie

Notre service de messagerie est géré par IONOS.
